Translation instructions

Download the package above and find the .mo file for your theme and language. Next, create a /languages directory within your theme directory (eg. linen_pro/languages) or your PRO child theme (eg. linen_pro_child/languages) directory and copy the .mo into the new directory. Finally, define your language in your wp-config.php file.

Missing? Create your own!

All of our themes include a .po file by default. The file may be used with a program like poEdit to generate a set of translation files for the theme in your language.
